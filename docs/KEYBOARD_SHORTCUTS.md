# Todone Keyboard Shortcuts

## Global
- `Cmd/Ctrl + K`: Open command palette / quick find
- `Escape`: Close modals and command palette
- `Cmd/Ctrl + S`: Save (planned)
- `Cmd/Ctrl + /`: Keyboard shortcuts reference (planned)

## Task Actions
- `Q`: Quick add task (planned)
- `A`: Add task at end of list (planned)
- `Shift + A`: Add task at top of list (planned)
- `Cmd/Ctrl + Enter`: Complete selected task (planned)
- `1` - `4`: Set priority P1-P4 (planned)
- `T`: Set due date to today (planned)
- `M`: Set due date to tomorrow (planned)
- `W`: Set due date to next week (planned)

## Navigation (Go Menu)
- `G then I`: Go to Inbox (planned)
- `G then T`: Go to Today (planned)
- `G then U`: Go to Upcoming (planned)

## Selection & Editing
- `Arrow keys`: Navigate tasks (planned)
- `Cmd/Ctrl + Up/Down`: Move task (planned)
- `Cmd/Ctrl + ]`: Indent (planned)
- `Cmd/Ctrl + [`: Outdent (planned)

## Search
- `/`: Focus search input (planned)
